# VibeVoice-ASR Colab-notitieboekje

Deze notebook transcribeert audio- of videobestanden.

## Kenmerken

- **Gratis te gebruiken:** Deze notebook maakt gebruik van [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), dat Microsoft op 21 januari 2026 open source heeft gemaakt, samen met de gratis GPU runtime van Google Colab. Om het model bruikbaar te maken op gratis GPU's, gebruikt deze notebook een gekwantiseerd VibeVoice-ASR-model.
- **Geen vaste limiet voor de bestandsgrootte aan de notebookzijde:** Grote audio- en videobestanden kunnen worden verwerkt door ze in segmenten te splitsen.
- **Batchverwerking:** Meerdere audio- en videobestanden kunnen in één run worden verwerkt.
- **Google Drive ondersteuning:** Bestanden kunnen rechtstreeks worden geüpload, of audio-/videobestanden die zijn opgeslagen in Google Drive kunnen worden verwerkt.
- **Volledig online:** Downloads en verwerking worden uitgevoerd op Google Colab. Er worden geen lokale computerbronnen gebruikt.
- **Meerdere uitvoerformaten:** De notebook voert het originele JSON transcript en de afgeleide TXT, CSV, Markdown, Excel en SRT ondertitelbestanden uit.

## Indeling van de opslagplaats

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Het sjabloonnotitieblok is het bronnotitieboekje en bevat tijdelijke aanduidingen zoals
`{{settings.input_mode_help}}`. Landinstellingsbestanden bieden taalspecifieke waarden
voor Colab-gerichte kopieën zoals `# @title`, `# @markdown`, opmerkingen en geselecteerde
gebruikersberichten. Elke landinstelling bevat de volledige set tijdelijke aanduidingen uit de
Engelse bronlocatie.

Gegenereerde notebooks worden vastgelegd, zodat Colab-links stabiel blijven.

## Bouwen

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Vertalingen vernieuwen

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` vertaalt automatisch de volledige Engelse landinstelling naar de
andere landinstellingsbestanden met behoud van codetokens zoals `INPUT_MODE`,
`Google Drive`, `# @markdown` en model-ID's. Vertaald exemplaar beoordelen
vóór een formele vrijlating.

## Cheques

```bash
python scripts/check_notebooks.py
```

De CI-workflowcontroles of gegenereerde notitieboekjes up-to-date zijn, zijn niet opgeslagen
celuitvoer, geparseerd als JSON, bevat geen onopgeloste tijdelijke aanduidingen, en die code
cellen zijn geldige Python-syntaxis.

## Wat CI kan testen

Deze opslagplaats kan de notebook testen als een distribueerbaar artefact:

- genereren op basis van de sjabloon- en landinstellingenbestanden
- notitieboekje JSON geldigheid
- lege uitvoeringsuitgangen
- Python-syntaxis in codecellen
- voor de hand liggende toevallige geheimen
- vastgelegde gegenereerde notitieboekjes die overeenkomen met de sjabloon

Het volledige transcriptiepad vereist Colab-specifieke API's, grote modeldownloads,
GPU beschikbaarheid en door de gebruiker geleverde mediabestanden. Dat moet worden behandeld als een
handmatige rooktest of een aparte GPU/Colab-validatieworkflow.
