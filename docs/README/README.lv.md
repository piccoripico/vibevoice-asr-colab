# VibeVoice-ASR Colab piezīmju grāmatiņa

Šis piezīmjdators pārraksta audio vai video failus.

## Funkcijas

- **Bezmaksas lietošana:** šajā piezīmjdatorā tiek izmantots [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), kuru Microsoft 2026. gada 21. janvārī nodrošināja atvērtā koda versija, kopā ar Google Colab bezmaksas GPU izpildlaiku. Lai modeli varētu izmantot bezmaksas GPU, šajā piezīmjdatorā tiek izmantots kvantizēts VibeVoice-ASR modelis.
- **Nav fiksēta piezīmjdatora faila lieluma ierobežojuma:** lielus audio un video failus var apstrādāt, sadalot tos segmentos.
- **Pakešapstrāde:** vienā reizē var apstrādāt vairākus audio un video failus.
- **Google Drive atbalsts:** Failus var augšupielādēt tieši vai var apstrādāt audio/video failus, kas saglabāti Google Drive.
- **Pilnībā tiešsaistē:** lejupielāde un apstrāde tiek veikta Google Colab. Vietējie skaitļošanas resursi netiek izmantoti.
- **Vairāki izvades formāti:** piezīmju grāmatiņa izvada oriģinālo JSON atšifrējumu un atvasinātos TXT, CSV, Markdown, Excel un SRT subtitru failus.

## Repozitorija izkārtojums

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Veidņu piezīmju grāmatiņa ir avota piezīmju grāmatiņa, un tajā ir vietturi, piemēram,
`{{settings.input_mode_help}}`. Lokalizācijas faili nodrošina valodai specifiskas vērtības
uz Colab vērstai kopijai, piemēram, `# @title`, `# @markdown`, komentāriem un atlasītajiem
lietotāju ziņas. Katrā lokalizācijā ir pilna vietturu kopa no
Angļu avota lokalizācija.

Ģenerētie bloknoti ir izveidoti, tāpēc Colab saites paliek stabilas.

## Veidot

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Atsvaidzināt tulkojumus

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` mašīntulko visu angļu valodas lokalizāciju valodā
citus lokalizācijas failus, vienlaikus saglabājot koda marķierus, piemēram, `INPUT_MODE`,
`Google Drive`, `# @markdown` un modeļa identifikatorus. Pārskatiet tulkoto kopiju
pirms oficiālas atbrīvošanas.

## Pārbaudes

```bash
python scripts/check_notebooks.py
```

CI darbplūsmas pārbaudes, vai ģenerētie piezīmjdatori ir atjaunināti, nav saglabāti
šūnu izvadi, parsēt kā JSON, nesatur neatrisinātus vietturus, un šis kods
šūnām ir derīga Python sintakse.

## Ko CI var pārbaudīt

Šajā krātuvē var pārbaudīt piezīmju grāmatiņu kā izplatāmu artefaktu:

- ģenerēšana no veidnes un lokalizācijas failiem
- piezīmju grāmatiņa JSON derīguma termiņš
- tukšas izpildes izvades
- Python sintakse koda šūnās
- acīmredzami nejauši noslēpumi
- izveidotās piezīmju grāmatiņas, kas atbilst veidnei

Pilnam transkripcijas ceļam ir nepieciešamas Colab specifiskas API, lielas modeļu lejupielādes,
GPU pieejamība un lietotāja nodrošināti multivides faili. Tas būtu jāuztver kā a
manuāla dūmu pārbaude vai atsevišķa GPU/Colab validācijas darbplūsma.
