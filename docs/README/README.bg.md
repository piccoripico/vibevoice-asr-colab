# VibeVoice-ASR Colab Бележник

Този бележник транскрибира аудио или видео файлове.

## Характеристики

- **Безплатен за използване:** Този преносим компютър използва [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), който Microsoft отвори с отворен код на 21 януари 2026 г., заедно с безплатното GPU време за изпълнение на Google Colab. За да направи модела използваем на безплатни графични процесори, този ноутбук използва квантован VibeVoice-ASR модел.
- **Няма фиксирано ограничение за размера на файла от страната на преносимия компютър:** Големите аудио и видео файлове могат да се обработват чрез разделянето им на сегменти.
- **Пакетна обработка:** Множество аудио и видео файлове могат да бъдат обработени наведнъж.
- **Google Drive поддръжка: ** Файловете могат да се качват директно или могат да се обработват аудио/видео файлове, съхранени в Google Drive.
- **Напълно онлайн:** Изтеглянията и обработката се извършват на Google Colab. Не се използват локални изчислителни ресурси.
- **Множество изходни формати:** Бележникът извежда оригиналния JSON препис и производни TXT, CSV, Markdown, Excel и SRT файлове със субтитри.

## Оформление на хранилището

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Шаблонният бележник е изходният бележник и съдържа контейнери като
`{{settings.input_mode_help}}`. Локалните файлове предоставят специфични за езика стойности
за копие, обърнато към Colab, като `# @title`, `# @markdown`, коментари и избрани
потребителски съобщения. Всеки локал съдържа пълния набор от контейнери от
Английски език на източника.

Генерираните бележници са ангажирани, така че връзките на Colab остават стабилни.

## Изграждане

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Опресняване на преводите

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` машинно превежда пълния английски език на
други локални файлове, като същевременно запазва кодови токени като `INPUT_MODE`,
`Google Drive`, `# @markdown` и идентификатори на модели. Прегледайте преведеното копие
преди официално освобождаване.

## Проверки

```bash
python scripts/check_notebooks.py
```

Работният поток на CI проверява дали генерираните бележници са актуални, нямат запазени
клетъчни изходи, анализирани като JSON, не съдържат неразрешени контейнери и този код
клетките са валиден синтаксис на Python.

## Какво CI може да тества

Това хранилище може да тества бележника като артефакт за разпространение:

- генериране от файловете на шаблона и локала
- тетрадка JSON валидност
- празни изходи за изпълнение
- Синтаксис на Python в кодови клетки
- очевидни случайни тайни
- ангажирани генерирани тетрадки, съответстващи на шаблона

Пълният път на транскрипция изисква специфични за Colab API, големи изтегляния на модели,
GPU наличност и медийни файлове, предоставени от потребителя. Това трябва да се третира като a
ръчен тест за дим или отделен работен процес за валидиране GPU/Colab.
