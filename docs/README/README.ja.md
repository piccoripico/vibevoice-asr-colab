# VibeVoice-ASR Colab ノートブック

このノートブックは、音声ファイルまたは動画ファイルを文字起こしします。

## 特徴

- **無料で利用可能:** このノートブックは、Microsoft が 2026年1月21日にオープンソース化した [VibeVoice-ASR](https://github.com/microsoft/VibeVoice) と、Google Colab の無料GPUランタイムを使います。無料GPUでも扱いやすいように、量子化された VibeVoice-ASR モデルを使います。
- **ノートブック側の固定ファイルサイズ制限なし:** 大きな音声・動画ファイルも、セグメントに分割して処理できます。
- **バッチ処理:** 複数の音声・動画ファイルを1回の実行で処理できます。
- **Google Drive対応:** ファイルを直接アップロードすることも、Google Drive内の音声・動画ファイルを処理することもできます。
- **完全オンライン:** ダウンロードと処理は Google Colab 上で行われます。ローカルPCの計算資源は使いません。
- **複数の出力形式:** 元のJSON形式の文字起こし結果に加えて、TXT、CSV、Markdown、Excel、SRT字幕ファイルを出力します。

## リポジトリのレイアウト

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

テンプレート ノートブックはソース ノートブックであり、次のようなプレースホルダーが含まれています。
`{{settings.input_mode_help}}`。ロケール ファイルは言語固有の値を提供します
`# @title`、`# @markdown`、コメント、選択されたものなどの Colab 向きのコピーの場合
ユーザーメッセージ。すべてのロケールには、
英語のソースロケール。

生成されたノートブックはコミットされるため、Colab リンクは安定した状態に保たれます。

## 建てる

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## 翻訳を更新する

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` は、完全な英語ロケールを次の言語に機械翻訳します。
`INPUT_MODE` などのコード トークンを保持しながら、他のロケール ファイルを保存します。
`Google Drive`、`# @markdown`、およびモデル識別子。翻訳されたコピーを確認する
正式リリース前に。

## 小切手

```bash
python scripts/check_notebooks.py
```

CI ワークフローは、生成されたノートブックが最新であること、保存されていないことをチェックします。
セル出力は JSON として解析され、未解決のプレースホルダーが含まれておらず、そのコードが
セルは有効な Python 構文です。

## CI でテストできること

このリポジトリは、ノートブックを配布可能なアーティファクトとしてテストできます。

- テンプレートとロケール ファイルからの生成
- ノートブックJSONの有効性
- 空の実行出力
- コードセル内のPython構文
- 明らかな偶然の秘密
- テンプレートに一致する生成されたノートブックをコミットしました

完全な転写パスには、Colab 固有の API、大規模なモデルのダウンロード、
GPU の可用性、およびユーザーが提供するメディア ファイル。それはとして扱われるべきです
手動の煙テストまたは別の GPU/Colab 検証ワークフロー。
