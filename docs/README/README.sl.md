# VibeVoice-ASR Beležnica Colab

Ta prenosni računalnik prepisuje avdio ali video datoteke.

## Lastnosti

- **Brezplačna uporaba:** Ta prenosnik uporablja [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), ki ga je Microsoft odprl 21. januarja 2026, skupaj z Google Colab Google Colab Google Colab izvajalnim okoljem. Da bi bil model uporaben na brezplačnih grafičnih procesorjih, ta prenosnik uporablja kvantiziran model VibeVoice-ASR.
- **Brez fiksne omejitve velikosti datoteke na strani prenosnega računalnika:** Velike zvočne in video datoteke je mogoče obdelati tako, da jih razdelite na segmente.
- **Paketna obdelava:** V enem zagonu je mogoče obdelati več avdio in video datotek.
- **Google Drive podpora:** Datoteke je mogoče naložiti neposredno ali obdelati avdio/video datoteke, shranjene v Google Drive.
- **V celoti na spletu:** Prenosi in obdelava se izvajajo na Google Colab. Lokalni računalniški viri se ne uporabljajo.
- **Več izhodnih formatov:** Beležnica izpiše izvirni JSON prepis in izpeljane TXT, CSV, Markdown, Excel in SRT datoteke s podnapisi.

## Postavitev skladišča

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Zvezek s predlogo je izvorni zvezek in vsebuje ogradne oznake, kot je npr
`{{settings.input_mode_help}}`. Področne datoteke zagotavljajo vrednosti, specifične za jezik
za kopijo, usmerjeno v Colab, kot je `# @title`, `# @markdown`, komentarji in izbrani
uporabniška sporočila. Vsaka področna nastavitev vsebuje celoten nabor nadomestnih znakov iz
angleški izvorni jezik.

Ustvarjeni zvezki so predani, tako da povezave Colab ostanejo stabilne.

## Zgradite

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Osveži prevode

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` strojno prevede celotno angleško področno nastavitev v
druge področne datoteke, medtem ko ohranja kodne žetone, kot je `INPUT_MODE`,
`Google Drive`, `# @markdown` in identifikatorji modela. Preglejte prevedeno kopijo
pred uradno izpustitvijo.

## Čeki

```bash
python scripts/check_notebooks.py
```

Potek dela CI preveri, ali so ustvarjeni zvezki posodobljeni in nimajo shranjenih
izhodi celic, razčlenjeni kot JSON, ne vsebujejo nerazrešenih nadomestnih mest in ta koda
celice so veljavna sintaksa Python.

## Kaj lahko testira CI

To skladišče lahko preizkusi prenosni računalnik kot artefakt, ki ga je mogoče distribuirati:

- ustvarjanje iz datotek predloge in področnih nastavitev
- veljavnost zvezka JSON
- prazni izhodi izvajanja
- Sintaksa Python v celicah kode
- očitne naključne skrivnosti
- committed generirani zvezki, ki se ujemajo s predlogo

Celotna pot prepisa zahteva API-je, specifične za Colab, velike prenose modelov,
GPU razpoložljivost in medijske datoteke, ki jih zagotovijo uporabniki. To je treba obravnavati kot a
ročni preizkus dima ali ločen potek dela GPU/Colab.
