# VibeVoice-ASR Colab дэвтэр

Энэ дэвтэр нь аудио эсвэл видео файлуудыг хөрвүүлдэг.

## Онцлогууд

- **Хэрэглэхэд үнэ төлбөргүй:** Энэхүү дэвтэр нь 2026 оны 1-р сарын 21-нд Microsoft-ын нээлттэй эх сурвалжтай болсон [VibeVoice-ASR](https://github.com/microsoft/VibeVoice)-г Google Colab-ийн үнэгүй GPU ажиллах цагийн хамт ашигладаг. Загварыг үнэ төлбөргүй GPU дээр ашиглах боломжтой болгохын тулд энэ дэвтэр VibeVoice-ASR загварыг ашигладаг.
- **Тэмдэглэлийн дэвтэр дээрх файлын хэмжээ хязгааргүй:** Том хэмжээний аудио болон видео файлуудыг хэсэг болгон хувааж боловсруулж болно.
- **Багц боловсруулалт:** Олон аудио болон видео файлуудыг нэг удаа боловсруулах боломжтой.
- **Google Drive дэмжлэг:** Файлуудыг шууд байршуулах эсвэл Google Drive-д хадгалагдсан аудио/видео файлуудыг боловсруулах боломжтой.
- **Бүрэн онлайн:** Татаж авах, боловсруулах ажлыг Google Colab дээр гүйцэтгэдэг. Орон нутгийн тооцоолох нөөц ашиглагдахгүй.
- **Олон гаралтын формат:** Тэмдэглэлийн дэвтэр нь эх JSON транскриптийг гаргаж, TXT, CSV, Markdown, Excel болон SRT хадмал файлуудыг гаргадаг.

## Репозиторийн бүтэц

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Загварын дэвтэр нь эх дэвтэр бөгөөд гэх мэт орлуулагчийг агуулна
`{{settings.input_mode_help}}`. Локал файлууд нь хэлний тусгай утгыг өгдөг
`# @title`, `# @markdown`, сэтгэгдэл, сонгосон зэрэг Colab-д зориулсан хуулбар
хэрэглэгчийн мессежүүд. Хэлэлцүүлэг бүр нь -аас авсан бүрэн орлуулагчийг агуулна
Англи хэлний эх сурвалж.

Үүсгэсэн тэмдэглэлийн дэвтэр нь үнэн зөв байх тул Colab холбоосууд тогтвортой байх болно.

## барих

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Орчуулгыг сэргээх

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` машин-англи хэлийг бүтэн хэл рүү хөрвүүлдэг
`INPUT_MODE` гэх мэт кодын токенуудыг хадгалахын зэрэгцээ бусад локал файлууд,
`Google Drive`, `# @markdown`, загвар танигч. Орчуулсан хуулбарыг хянана уу
албан ёсоор гарахаас өмнө.

## Шалгалтууд

```bash
python scripts/check_notebooks.py
```

CI ажлын урсгал нь үүсгэсэн тэмдэглэлийн дэвтэр шинэчлэгдсэн, хадгалагдсан зүйл байхгүй эсэхийг шалгадаг
JSON гэж задлан шинжлэх нүдний гаралт нь шийдэгдээгүй орлуулагч агуулаагүй бөгөөд код
эсүүд нь хүчинтэй Python синтакс юм.

## CI юуг шалгаж чадах вэ?

Энэ агуулах нь тэмдэглэлийн дэвтэрийг тарааж болох олдвор болгон шалгах боломжтой:

- загвар болон локал файлуудаас үүсгэх
- тэмдэглэлийн дэвтэр JSON хүчинтэй
- хоосон гүйцэтгэлийн гаралт
- Кодын нүдн дэх Python синтакс
- илт санамсаргүй нууцууд
- загварт тохирсон дэвтэр үүсгэсэн

Транскрипцийн бүрэн зам нь Colab-д зориулсан API, том загвар татаж авах,
GPU бэлэн байдал, хэрэглэгчийн өгсөн медиа файлууд. Үүнийг а гэж үзэх ёстой
гар аргаар утааны туршилт эсвэл тусдаа GPU/Colab баталгаажуулалтын ажлын урсгал.
