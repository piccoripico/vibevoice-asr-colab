# VibeVoice-ASR Colab Notaboek

Hierdie notaboek transkribeer oudio- of videolêers.

## Kenmerke

- **Gratis om te gebruik:** Hierdie notaboek gebruik [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), wat Microsoft op 21 Januarie 2026 oopbron verskaf het, tesame met Google Colab se gratis GPU looptyd. Om die model op gratis GPU's bruikbaar te maak, gebruik hierdie notaboek 'n gekwantiseerde VibeVoice-ASR-model.
- **Geen vaste lêergroottelimiet aan die notaboekkant nie:** Groot oudio- en videolêers kan verwerk word deur dit in segmente te verdeel.
- **Joernaalverwerking:** Veelvuldige oudio- en videolêers kan in een lopie verwerk word.
- **Google Drive ondersteuning:** Lêers kan direk opgelaai word, of oudio-/videolêers wat in Google Drive gestoor is, kan verwerk word.
- **Ten volle aanlyn:** Aflaaie en verwerking word uitgevoer op Google Colab. Geen plaaslike rekenaarhulpbronne word gebruik nie.
- **Verskeie uitvoerformate:** Die notaboek voer die oorspronklike JSON transkripsie en afgeleide TXT, CSV, Markdown, Excel, en SRT ondertitellêers uit.

## Bewaarplekuitleg

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Die sjabloon notaboek is die bron notaboek en bevat plekhouers soos
`{{settings.input_mode_help}}`. Locale-lêers verskaf taalspesifieke waardes
vir Colab-gesig kopie soos `# @title`, `# @markdown`, opmerkings, en geselekteerde
gebruiker boodskappe. Elke plek bevat die volledige plekhouerstel van die
Engelse bron-lokaal.

Gegenereerde notaboeke is verbind sodat Colab-skakels stabiel bly.

## Bou

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Herlaai vertalings

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` masjien-vertaal die volledige Engelse taal in die
ander locale lêers terwyl kodetokens soos `INPUT_MODE` bewaar word,
`Google Drive`, `# @markdown`, en model identifiseerders. Hersien vertaalde kopie
voor 'n formele vrylating.

## Tjeks

```bash
python scripts/check_notebooks.py
```

Die CI-werkvloeikontroles wat gegenereerde notaboeke op datum is, is nie gestoor nie
seluitvoere, ontleed as JSON, bevat geen onopgeloste plekhouers nie, en daardie kode
selle is geldige Python-sintaksis.

## Wat CI kan toets

Hierdie bewaarplek kan die notaboek as 'n verspreibare artefak toets:

- generasie vanaf die sjabloon en locale lêers
- notaboek JSON geldigheid
- leë uitvoering uitsette
- Python-sintaksis in kodeselle
- ooglopende toevallige geheime
- toegewyde gegenereerde notaboeke wat ooreenstem met die sjabloon

Die volledige transkripsiepad vereis Colab-spesifieke API's, groot model aflaaie,
GPU beskikbaarheid, en gebruikerverskafde medialêers. Dit moet as 'n
handmatige rooktoets of 'n aparte GPU/Colab-bekragtigingswerkvloei.
