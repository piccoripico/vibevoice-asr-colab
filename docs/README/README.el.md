# VibeVoice-ASR Τετράδιο Colab

Αυτό το σημειωματάριο μεταγράφει αρχεία ήχου ή βίντεο.

## Χαρακτηριστικά

- **Δωρεάν χρήση:** Αυτό το σημειωματάριο χρησιμοποιεί το [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), το οποίο η Microsoft δημιούργησε ανοιχτό κώδικα στις 21 Ιανουαρίου 2026, μαζί με τον δωρεάν χρόνο εκτέλεσης Google Colab του GPU. Για να γίνει το μοντέλο χρησιμοποιήσιμο σε δωρεάν GPU, αυτός ο φορητός υπολογιστής χρησιμοποιεί ένα κβαντισμένο μοντέλο VibeVoice-ASR.
- **Χωρίς σταθερό όριο μεγέθους αρχείου από την πλευρά του σημειωματαρίου:** Μεγάλα αρχεία ήχου και βίντεο μπορούν να υποβληθούν σε επεξεργασία χωρίζοντάς τα σε τμήματα.
- **Μαζική επεξεργασία:** Πολλά αρχεία ήχου και βίντεο μπορούν να υποβληθούν σε επεξεργασία σε μία εκτέλεση.
- **Google Drive υποστήριξη:** Τα αρχεία μπορούν να μεταφορτωθούν απευθείας ή τα αρχεία ήχου/βίντεο που είναι αποθηκευμένα στο Google Drive μπορούν να υποβληθούν σε επεξεργασία.
- **Πλήρως online:** Οι λήψεις και η επεξεργασία πραγματοποιούνται στο Google Colab. Δεν χρησιμοποιούνται τοπικοί υπολογιστικοί πόροι.
- **Πολλαπλές μορφές εξόδου:** Το σημειωματάριο εξάγει την αρχική μεταγραφή JSON και τα προκύπτοντα αρχεία υπότιτλων TXT, CSV, Markdown, Excel και SRT.

## Διάταξη αποθετηρίου

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Το σημειωματάριο προτύπου είναι το σημειωματάριο πηγής και περιέχει σύμβολα κράτησης θέσης όπως
`{{settings.input_mode_help}}`. Τα τοπικά αρχεία παρέχουν τιμές για συγκεκριμένες γλώσσες
για αντίγραφο που βλέπει Colab όπως `# @title`, `# @markdown`, σχόλια και επιλεγμένα
μηνύματα χρήστη. Κάθε τοπική ρύθμιση περιέχει το πλήρες σύνολο συμβόλων κράτησης θέσης από το
Αγγλική πηγή τοπικής γλώσσας.

Τα σημειωματάρια που δημιουργούνται είναι δεσμευμένα, ώστε οι σύνδεσμοι Colab να παραμένουν σταθεροί.

## Οικοδομώ

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Ανανέωση μεταφράσεων

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

Το `translate_full_locales.py` μηχανή-μεταφράζει την πλήρη αγγλική γλώσσα στο
άλλα αρχεία τοπικών ρυθμίσεων ενώ διατηρούνται διακριτικά κώδικα όπως `INPUT_MODE`,
`Google Drive`, `# @markdown` και αναγνωριστικά μοντέλου. Ελέγξτε το μεταφρασμένο αντίγραφο
πριν από την επίσημη απελευθέρωση.

## Έλεγχοι

```bash
python scripts/check_notebooks.py
```

Οι έλεγχοι ροής εργασιών CI ότι τα σημειωματάρια που δημιουργούνται είναι ενημερωμένα, δεν έχουν αποθηκευτεί
Οι έξοδοι κελιών, αναλύονται ως JSON, δεν περιέχουν μη επιλυμένα σύμβολα κράτησης θέσης και αυτός ο κώδικας
Τα κελιά είναι έγκυρη σύνταξη Python.

## Τι μπορεί να δοκιμάσει το CI

Αυτό το αποθετήριο μπορεί να δοκιμάσει το σημειωματάριο ως τεχνούργημα με δυνατότητα διανομής:

- δημιουργία από το πρότυπο και τα αρχεία τοπικών ρυθμίσεων
- σημειωματάριο JSON εγκυρότητα
- κενές έξοδοι εκτέλεσης
- Σύνταξη Python σε κελιά κώδικα
- προφανή τυχαία μυστικά
- δεσμευμένα δημιουργημένα σημειωματάρια που ταιριάζουν με το πρότυπο

Η πλήρης διαδρομή μεταγραφής απαιτεί API ειδικά για το Colab, μεγάλες λήψεις μοντέλων,
Διαθεσιμότητα GPU και αρχεία πολυμέσων που παρέχονται από τον χρήστη. Αυτό θα πρέπει να αντιμετωπίζεται ως α
χειροκίνητη δοκιμή καπνού ή ξεχωριστή ροή εργασιών επικύρωσης GPU/Colab.
