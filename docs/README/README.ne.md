## सुविधाहरू

- **प्रयोग गर्न नि:शुल्क:** यो नोटबुकले [VibeVoice-ASR](https://github.com/microsoft/VibeVoice) प्रयोग गर्दछ, जुन माइक्रोसफ्टले जनवरी 21, 2026 मा Google Colab को नि:शुल्क GPU रनटाइमसँग ओपन सोर्स गरेको थियो। मोडेललाई नि:शुल्क GPU मा प्रयोगयोग्य बनाउन, यो नोटबुकले क्वान्टाइज्ड VibeVoice-ASR मोडेल प्रयोग गर्दछ।
- **कुनै निश्चित नोटबुक-साइड फाइल साइज सीमा छैन:** ठूला अडियो र भिडियो फाइलहरूलाई खण्डहरूमा विभाजन गरेर प्रशोधन गर्न सकिन्छ।
- ** ब्याच प्रशोधन: ** बहु अडियो र भिडियो फाइलहरू एक रनमा प्रशोधन गर्न सकिन्छ।
- **Google Drive समर्थन: ** फाइलहरू सीधा अपलोड गर्न सकिन्छ, वा Google Drive मा भण्डारण गरिएका अडियो/भिडियो फाइलहरू प्रशोधन गर्न सकिन्छ।
- **पूरै अनलाइन:** डाउनलोड र प्रशोधन Google Colab मा गरिन्छ। स्थानीय कम्प्युटिङ स्रोतहरू प्रयोग गरिएको छैन।
- **बहु आउटपुट ढाँचाहरू:** नोटबुकले मूल JSON ट्रान्सक्रिप्ट र व्युत्पन्न TXT, CSV, मार्कडाउन, एक्सेल, र SRT उपशीर्षक फाइलहरू आउटपुट गर्दछ।

## भण्डार लेआउट

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

टेम्प्लेट नोटबुक स्रोत नोटबुक हो र प्लेसहोल्डरहरू समावेश गर्दछ जस्तै
`{{settings.input_mode_help}}`। स्थानीय फाइलहरूले भाषा-विशिष्ट मानहरू प्रदान गर्दछ
`# @title`, `# @markdown`, टिप्पणीहरू, र चयन गरिएको कोलाब-फेसिङ प्रतिलिपिका लागि
प्रयोगकर्ता सन्देशहरू। प्रत्येक लोकेलले बाट पूर्ण प्लेसहोल्डर सेट समावेश गर्दछ
अंग्रेजी स्रोत लोकेल।

उत्पन्न गरिएका नोटबुकहरू प्रतिबद्ध छन् त्यसैले Colab लिङ्कहरू स्थिर रहन्छन्।

## निर्माण गर्नुहोस्

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## अनुवादहरू ताजा गर्नुहोस्

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` मेसिनले पूर्ण अङ्ग्रेजी लोकेललाई मा अनुवाद गर्छ
कोड टोकनहरू सुरक्षित गर्दा अन्य लोकेल फाइलहरू जस्तै `INPUT_MODE`,
`Google Drive`, `# @markdown`, र मोडेल पहिचानकर्ताहरू। अनुवादित प्रतिलिपि समीक्षा गर्नुहोस्
औपचारिक विमोचन अघि।

## चेक

```bash
python scripts/check_notebooks.py
```

CI कार्यप्रवाह जाँचहरू कि उत्पन्न नोटबुकहरू अद्यावधिक छन्, कुनै पनि बचत गरिएको छैन
सेल आउटपुटहरू, JSON को रूपमा पार्स गर्नुहोस्, कुनै समाधान नगरिएको प्लेसहोल्डरहरू, र त्यो कोड
कक्षहरू मान्य पाइथन सिन्ट्याक्स हुन्।

## CI ले के परीक्षण गर्न सक्छ

यो भण्डारले नोटबुकलाई वितरणयोग्य कलाकृतिको रूपमा परीक्षण गर्न सक्छ:

- टेम्प्लेट र लोकेल फाइलहरूबाट उत्पादन
- नोटबुक JSON वैधता
- खाली निष्पादन आउटपुटहरू
- कोड कक्षहरूमा पाइथन सिन्ट्याक्स
- स्पष्ट आकस्मिक रहस्य
