# VibeVoice-ASR Colab Jegyzetfüzet

Ez a notebook hang- és videofájlokat ír át.

## Jellemzők

- **Ingyen használható:** Ez a notebook a [VibeVoice-ASR](https://github.com/microsoft/VibeVoice) kódot használja, amelyet a Microsoft nyílt forráskódú 2026. január 21-én, valamint a Google Colab ingyenes GPU futtatókörnyezetét. Annak érdekében, hogy a modell ingyenes GPU-kon is használható legyen, ez a notebook kvantált VibeVoice-ASR modellt használ.
- **Nincs rögzített notebookoldali fájlméretkorlát:** A nagy audio- és videofájlok szegmensekre bontással dolgozhatók fel.
- **Kötegelt feldolgozás:** Több audio- és videofájl is feldolgozható egy futással.
- **Google Drive támogatás:** A fájlok közvetlenül feltölthetők, vagy a Google Drive címen tárolt audio-/videofájlok feldolgozhatók.
- **Teljesen online:** A letöltések és a feldolgozás a Google Colab címen történik. Nem használnak helyi számítási erőforrásokat.
- **Több kimeneti formátum:** A notebook az eredeti JSON átiratot és a származtatott TXT, CSV, Markdown, Excel és SRT feliratfájlokat adja ki.

## Adatt?r fel?p?t?se

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

A sablon jegyzetfüzet a forrásjegyzetfüzet, és helyőrzőket tartalmaz, mint pl
`{{settings.input_mode_help}}`. A nyelvi fájlok nyelvspecifikus értékeket biztosítanak
a Colab felé néző példányokhoz, például `# @title`, `# @markdown`, megjegyzésekhez és kiválasztott
felhasználói üzenetek. Minden területi beállítás tartalmazza a teljes helyőrző-készletet a
Angol forrásnyelv.

A generált jegyzetfüzetek elkötelezettek, így a Colab-linkek stabilak maradnak.

## Épít

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Fordítások frissítése

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` gépi lefordítja a teljes angol nyelvterületet a
más területi beállítások fájlok, miközben megőrzik a kódtokeneket, például `INPUT_MODE`,
`Google Drive`, `# @markdown` és modellazonosítók. Tekintse át a lefordított példányt
hivatalos kiadás előtt.

## Ellenőrzések

```bash
python scripts/check_notebooks.py
```

A CI-munkafolyamat ellenőrzi, hogy a generált jegyzetfüzetek naprakészek-e, nincsenek-e mentve
a cellakimenetek JSON-ként értelmezik, nem tartalmaznak feloldatlan helyőrzőket, és ez a kód
A cellák érvényes Python szintaxissal rendelkeznek.

## Mit tesztelhet a CI

Ez a tároló képes tesztelni a notebookot terjeszthető műtermékként:

- generálás a sablon- és területi fájlokból
- notebook JSON érvényessége
- üres végrehajtási kimenetek
- Python szintaxis kódcellákban
- nyilvánvaló véletlen titkok
- elkötelezett generált jegyzetfüzetek, amelyek megfelelnek a sablonnak

A teljes átírási útvonalhoz Colab-specifikus API-k, nagyméretű modellletöltések,
GPU elérhetőség és a felhasználó által biztosított médiafájlok. Ezt úgy kell kezelni, mint a
kézi füstteszt vagy egy külön GPU/Colab érvényesítési munkafolyamat.
