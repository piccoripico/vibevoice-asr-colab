## Հատկանիշներ

- **Անվճար օգտագործման համար.** Այս նոթատետրն օգտագործում է [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), որը Microsoft-ը բաց կոդով տրամադրել է 2026թ. հունվարի 21-ին, Google Colab-ի անվճար GPU գործարկման ժամանակի հետ միասին: Մոդելը անվճար GPU-ներում օգտագործելի դարձնելու համար այս նոութբուքն օգտագործում է քվանտացված VibeVoice-ASR մոդելը:
- **Նոթատետրի կողմից ֆայլի չափի ֆիքսված սահմանափակում չկա.** Խոշոր աուդիո և վիդեո ֆայլերը կարող են մշակվել՝ դրանք բաժանելով հատվածների:
- **Խմբաքանակի մշակում.** Բազմաթիվ աուդիո և վիդեո ֆայլեր կարող են մշակվել մեկ վազքով:
- **Google Drive աջակցություն.** Ֆայլերը կարող են ուղղակիորեն վերբեռնվել, կամ Google Drive-ում պահվող աուդիո/վիդեո ֆայլերը կարող են մշակվել:
- **Լիովին առցանց.** Ներբեռնումները և մշակումը կատարվում են Google Colab հասցեով: Տեղական հաշվողական ռեսուրսներ չեն օգտագործվում:
- **Բազմաթիվ ելքային ձևաչափեր.** Նոթատետրը թողարկում է բնօրինակ JSON տառադարձությունը և ստացված TXT, CSV, Markdown, Excel և SRT ենթագրերի ֆայլերը:

## Պահեստի դասավորություն

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Կաղապարի նոթատետրը աղբյուրի նոթատետրն է և պարունակում է տեղապահեր, ինչպիսիք են
`{{settings.input_mode_help}}`. Տեղական ֆայլերը տալիս են լեզվին հատուկ արժեքներ
Colab-ի դեմքով պատճենելու համար, ինչպիսիք են `# @title`, `# @markdown`, մեկնաբանություններ և ընտրված
օգտվողի հաղորդագրությունները. Յուրաքանչյուր տեղաբաշխում պարունակում է տեղապահների ամբողջական հավաքածուն
Անգլերեն աղբյուրի տեղայնացում.

Ստեղծված նոթատետրերը նախատեսված են, որպեսզի Colab-ի հղումները մնան կայուն:

## Կառուցել

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Թարմացնել թարգմանությունները

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` մեքենա-թարգմանում է անգլերենի ամբողջական տեղանքը
այլ տեղային ֆայլեր՝ պահպանելով կոդերի նշանները, ինչպիսիք են `INPUT_MODE`,
`Google Drive`, `# @markdown` և մոդելի նույնացուցիչներ: Վերանայեք թարգմանված պատճենը

## Չեկեր

```bash
python scripts/check_notebooks.py
```

CI աշխատանքային հոսքի ստուգումները, որ ստեղծած նոթատետրերը թարմացված են, պահպանված չեն
բջջային ելքերը, վերլուծել որպես JSON, չեն պարունակում չլուծված տեղապահներ, և այդ կոդը
բջիջները վավեր Python շարահյուսություն են:

## Ինչ կարող է CI-ն ստուգել

Այս պահոցը կարող է փորձարկել նոթատետրը որպես բաշխվող արտեֆակտ.

- ձևանմուշից և տեղայնացման ֆայլերից առաջացում
- նոթատետր JSON վավերականությունը
- դատարկ կատարման ելքեր
- Python-ի շարահյուսությունը կոդի բջիջներում
- ակնհայտ պատահական գաղտնիքներ
- կատարել է ձևանմուշին համապատասխանող ստեղծած նոթատետրեր

Տառադարձման ամբողջական ուղին պահանջում է Colab-ին հատուկ API-ներ, մոդելների մեծ ներբեռնումներ,
GPU հասանելիություն և օգտագործողի կողմից տրամադրված մեդիա ֆայլեր: Դրան պետք է վերաբերվել որպես ա
ձեռքով ծխի փորձարկում կամ առանձին GPU/Colab վավերացման աշխատանքային հոսք:
