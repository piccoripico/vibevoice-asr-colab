# VibeVoice-ASR „Colab“ užrašų knygelė

Šis nešiojamasis kompiuteris perrašo garso arba vaizdo failus.

## Savybės

- **Nemokama naudoti:** Šiame nešiojamajame kompiuteryje naudojama [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), kurią „Microsoft“ sukūrė atvirojo kodo 2026 m. sausio 21 d., kartu su Google Colab nemokama GPU vykdymo laiku. Kad modelį būtų galima naudoti nemokamuose GPU, šis nešiojamasis kompiuteris naudoja kvantuotą VibeVoice-ASR modelį.
- **Nėra fiksuoto nešiojamojo kompiuterio failo dydžio apribojimo:** didelius garso ir vaizdo failus galima apdoroti padalijus juos į segmentus.
- **Paketinis apdorojimas:** vienu paleidimu galima apdoroti kelis garso ir vaizdo failus.
- **Google Drive palaikymas:** Failus galima įkelti tiesiogiai arba apdoroti Google Drive saugomus garso / vaizdo failus.
- **Visiškai prisijungę:** atsisiuntimai ir apdorojimas atliekami Google Colab. Nenaudojami jokie vietiniai skaičiavimo ištekliai.
- **Keli išvesties formatai:** Nešiojamasis kompiuteris išveda originalų JSON nuorašą ir išvestinius TXT, CSV, Markdown, Excel ir SRT subtitrų failus.

## Saugyklos išdėstymas

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Šablonų bloknotas yra šaltinio bloknotas, kuriame yra vietos rezervavimo ženklų, pvz
`{{settings.input_mode_help}}`. Lokalės failuose pateikiamos konkrečios kalbos reikšmės
„Colab“ skirtai kopijai, pvz., `# @title`, `# @markdown`, komentarams ir pasirinktoms
vartotojų žinutes. Kiekvienoje lokalėje yra visas rezervuotos vietos rinkinys iš
Anglų šaltinio lokalė.

Sugeneruotos užrašinės yra skirtos, todėl „Colab“ nuorodos išlieka stabilios.

## Sukurti

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Atnaujinti vertimus

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` mašina išverčia visą anglų kalbos lokalę į
kitus lokalės failus išsaugant kodo prieigos raktus, pvz., `INPUT_MODE`,
`Google Drive`, `# @markdown` ir modelio identifikatorius. Peržiūrėkite išverstą kopiją
prieš oficialų paleidimą.

## Patikros

```bash
python scripts/check_notebooks.py
```

CI darbo eigos patikrinimai, ar sukurti bloknotai yra atnaujinami, nėra išsaugoti
langelių išvestis, analizuoti kaip JSON, neturi neišspręstų rezervuotųjų ženklų ir tas kodas
langeliai yra tinkama Python sintaksė.

## Ką CI gali išbandyti

Ši saugykla gali išbandyti bloknotą kaip platinamą artefaktą:

- generavimas iš šablono ir lokalės failų
- bloknotas JSON galiojimas
- tušti vykdymo išėjimai
- Python sintaksė kodo ląstelėse
- akivaizdžios atsitiktinės paslaptys
- įsipareigojo sugeneruotas bloknotas, atitinkantis šabloną

Visam transkripcijos keliui reikia specialių „Colab“ API, didelių modelių atsisiuntimų,
GPU prieinamumas ir vartotojo pateikti medijos failai. Tai turėtų būti traktuojama kaip a
rankinis dūmų testas arba atskira GPU/Colab patvirtinimo darbo eiga.
