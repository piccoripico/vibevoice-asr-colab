# VibeVoice-ASR Colabi märkmik

See märkmik transkribeerib heli- või videofaile.

## Omadused

- **Tasuta kasutatav:** see sülearvuti kasutab [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), mille Microsoft on 21. jaanuaril 2026 avatud lähtekoodiga, koos Google Colab tasuta GPU käitusajaga. Mudeli kasutamiseks tasuta GPU-del kasutab see sülearvuti kvantiseeritud VibeVoice-ASR mudelit.
- **Fikseeritud sülearvutipoolse faili suuruse piiranguta:** suuri heli- ja videofaile saab töödelda, jagades need segmentideks.
- **Pakitöötlemine:** korraga saab töödelda mitut heli- ja videofaili.
- **Google Drive tugi:** Faile saab otse üles laadida või Google Drive salvestatud heli-/videofaile saab töödelda.
- **Täielikult võrgus:** allalaadimine ja töötlemine toimub aadressil Google Colab. Kohalikke arvutusressursse ei kasutata.
- **Mitu väljundvormingut:** sülearvuti väljastab algse JSON transkriptsiooni ja tuletatud TXT, CSV, Markdowni, Exceli ja SRT subtiitrite failid.

## Hoidla paigutus

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Mallmärkmik on lähtemärkmik ja sisaldab kohahoidjaid, näiteks
`{{settings.input_mode_help}}`. Lokaadifailid pakuvad keelepõhiseid väärtusi
for Colab-facing copy such as `# @title`, `# @markdown`, comments, and selected
kasutajate sõnumid. Every locale contains the complete placeholder set from the
Inglise allika lokaat.

Loodud märkmikud on pühendunud, nii et Colabi lingid püsivad stabiilsena.

## Ehita

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Värskenda tõlkeid

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` masintõlgib täieliku inglise keele lokaadi keelde
muud lokaadi failid, säilitades samas koodimärgid, nagu `INPUT_MODE`,
`Google Drive`, `# @markdown` ja mudeli identifikaatorid. Review translated copy
enne ametlikku vabastamist.

## Kontrollid

```bash
python scripts/check_notebooks.py
```

CI töövoo kontrollib, kas loodud märkmikud on ajakohased, neid pole salvestatud
lahtri väljundid, sõeluvad kujul JSON, ei sisalda lahendamata kohahoidjaid ja see kood
lahtrid on kehtiva Pythoni süntaksiga.

## Mida CI saab testida

See hoidla saab testida sülearvutit levitatava artefaktina:

- genereerimine malli- ja lokaadifailidest
- märkmik JSON kehtivus
- tühjad täitmisväljundid
- Pythoni süntaks koodilahtrites
- ilmsed juhuslikud saladused
- pühendunud loodud märkmikud, mis vastavad mallile

Täieliku transkriptsioonitee jaoks on vaja Colabi-spetsiifilisi API-sid, suuri mudelite allalaadimisi,
GPU kättesaadavus ja kasutaja pakutavad meediumifailid. That should be treated as a
käsitsi suitsutest või eraldi GPU/Colabi valideerimise töövoog.
