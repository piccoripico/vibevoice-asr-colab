# VibeVoice-ASR کولاب نوٹ بک

یہ نوٹ بک آڈیو یا ویڈیو فائلوں کو نقل کرتی ہے۔

## خصوصیات

- **استعمال کے لیے مفت:** یہ نوٹ بک [VibeVoice-ASR](https://github.com/microsoft/VibeVoice) کا استعمال کرتی ہے، جسے مائیکروسافٹ نے 21 جنوری 2026 کو Google Colab کے مفت GPU کے رن ٹائم کے ساتھ اوپن سورس کیا۔ ماڈل کو مفت GPUs پر قابل استعمال بنانے کے لیے، یہ نوٹ بک ایک کوانٹائزڈ VibeVoice-ASR ماڈل استعمال کرتی ہے۔
- **نوٹ بک سائیڈ فائل سائز کی کوئی مقررہ حد نہیں:** بڑی آڈیو اور ویڈیو فائلوں کو حصوں میں تقسیم کرکے ان پر کارروائی کی جاسکتی ہے۔
- **بیچ پروسیسنگ:** ایک رن میں متعدد آڈیو اور ویڈیو فائلوں پر کارروائی کی جا سکتی ہے۔
- **Google Drive سپورٹ:** فائلوں کو براہ راست اپ لوڈ کیا جا سکتا ہے، یا Google Drive میں محفوظ آڈیو/ویڈیو فائلوں پر کارروائی کی جا سکتی ہے۔
- **مکمل طور پر آن لائن:** ڈاؤن لوڈ اور پروسیسنگ Google Colab پر کی جاتی ہے۔ کوئی مقامی کمپیوٹنگ وسائل استعمال نہیں کیے جاتے ہیں۔
- **متعدد آؤٹ پٹ فارمیٹس:** نوٹ بک اصل JSON ٹرانسکرپٹ اور ماخوذ TXT، CSV، مارک ڈاؤن، ایکسل، اور SRT سب ٹائٹل فائلوں کو آؤٹ پٹ کرتی ہے۔

## ذخیرہ لے آؤٹ

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

ٹیمپلیٹ نوٹ بک سورس نوٹ بک ہے اور اس میں پلیس ہولڈرز ہوتے ہیں جیسے
`{{settings.input_mode_help}}`۔ مقامی فائلیں زبان کے ساتھ مخصوص اقدار فراہم کرتی ہیں۔
Colab کا سامنا کرنے والی کاپی کے لیے جیسے `# @title`، `# @markdown`، تبصرے، اور منتخب کردہ
صارف کے پیغامات. ہر لوکیل میں سے مکمل پلیس ہولڈر سیٹ ہوتا ہے۔
انگریزی ماخذ مقامی۔

جنریٹڈ نوٹ بکس پرعزم ہیں تاکہ Colab لنکس مستحکم رہیں۔

## تعمیر کریں۔

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## ترجمے کو تازہ کریں۔

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` مشین-مکمل انگریزی زبان کا ترجمہ کرتا ہے۔
کوڈ ٹوکن کو محفوظ کرتے ہوئے دیگر لوکل فائلیں جیسے `INPUT_MODE`،
`Google Drive`، `# @markdown`، اور ماڈل شناخت کار۔ ترجمہ شدہ کاپی کا جائزہ لیں۔
رسمی رہائی سے پہلے۔

## چیک کرتا ہے۔

```bash
python scripts/check_notebooks.py
```

CI ورک فلو چیک جو کہ تیار کردہ نوٹ بک اپ ٹو ڈیٹ ہیں، کوئی محفوظ نہیں ہے۔
سیل آؤٹ پٹس، JSON کے بطور پارس کریں، کوئی حل نہ شدہ پلیس ہولڈرز، اور وہ کوڈ
خلیات درست Python نحو ہیں۔

## CI کیا جانچ سکتا ہے۔

یہ ذخیرہ نوٹ بک کو قابل تقسیم نمونے کے طور پر جانچ سکتا ہے:

- ٹیمپلیٹ اور لوکیل فائلوں سے نسل
- نوٹ بک JSON درستگی
- خالی ایگزیکیوشن آؤٹ پٹس
- کوڈ سیلز میں ازگر کا نحو
- واضح حادثاتی راز
- ٹیمپلیٹ سے مماثل تیار شدہ نوٹ بک

مکمل ٹرانسکرپشن پاتھ کے لیے Colab کے لیے مخصوص APIs، بڑے ماڈل ڈاؤن لوڈز،
GPU دستیابی، اور صارف کی فراہم کردہ میڈیا فائلیں۔ اسے ایک کے طور پر علاج کیا جانا چاہئے
دستی دھواں ٹیسٹ یا علیحدہ GPU/کولاب کی توثیق کا ورک فلو۔
