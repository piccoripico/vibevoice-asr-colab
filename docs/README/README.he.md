# VibeVoice-ASR מחברת Colab

מחברת זו מתמללת קובצי אודיו או וידאו.

## תכונות

- **חופשי לשימוש:** מחברת זו משתמשת ב-[VibeVoice-ASR](https://github.com/microsoft/VibeVoice), ש-Microsoft הוציאה בקוד פתוח ב-21 בינואר 2026, יחד עם זמן הריצה החינמי של GPU. כדי להפוך את הדגם לשמיש על GPUs חינמיים, מחברת זו משתמשת בדגם VibeVoice-ASR כמותי.
- **אין מגבלה קבועה של גודל קבצים בצד המחברת:** ניתן לעבד קבצי אודיו ווידאו גדולים על ידי פיצולם למקטעים.
- **עיבוד אצווה:** ניתן לעבד קבצי אודיו ווידאו מרובים בהרצה אחת.
- **Google Drive תמיכה:** ניתן להעלות קבצים ישירות, או לעבד קבצי אודיו/וידאו המאוחסנים ב-Google Drive.
- **מקוון מלא:** ההורדות והעיבוד מתבצעים ב-Google Colab. לא נעשה שימוש במשאבי מחשוב מקומיים.
- **פורמטים מרובים של פלט:** המחברת מפלטת את התמליל המקורי JSON וקובצי הכתוביות TXT, CSV, Markdown, Excel ו-SRT.

## פריסת מאגר

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

מחברת התבנית היא מחברת המקור ומכילה מצייני מיקום כגון
`{{settings.input_mode_help}}`. קובצי מיקום מספקים ערכים ספציפיים לשפה
עבור עותק הפונה ל-Colab כגון `# @title`, `# @markdown`, הערות ונבחרות
הודעות משתמש. כל מקום מכיל את ערכת מציין המיקום המלאה מה-
מקום מקור אנגלי.

מחברות שנוצרו מחויבות כך שקישורי Colab נשארים יציבים.

## לִבנוֹת

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## רענן תרגומים

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` מתרגם במכונה את המקום האנגלי המלא ל-
קבצי מקומיים אחרים תוך שמירה על אסימוני קוד כגון `INPUT_MODE`,
`Google Drive`, `# @markdown`, ומזהי דגם. סקירת עותק מתורגם
לפני שחרור רשמי.

## צ'קים

```bash
python scripts/check_notebooks.py
```

בדיקות זרימת העבודה של CI שהמחברות שנוצרו מעודכנות, לא נשמרו
פלטי תא, נתח כ-JSON, אינם מכילים מצייני מיקום לא פתורים, והקוד הזה
תאים הם תחביר Python חוקי.

## מה CI יכול לבדוק

מאגר זה יכול לבדוק את המחברת כחפץ שניתן להפצה:

- יצירה מקבצי התבנית והמקום
- תוקף המחברת JSON
- פלטי ביצוע ריקים
- תחביר פייתון בתאי קוד
- סודות מקריים ברורים
- מחברות שנוצרו מחויבות התואמות לתבנית

נתיב התמלול המלא דורש ממשקי API ספציפיים ל-Colab, הורדות של דגמים גדולים,
זמינות GPU וקובצי מדיה שסופקו על ידי המשתמש. יש להתייחס לזה כאל א
בדיקת עשן ידנית או זרימת עבודה נפרדת של GPU/Colab.
