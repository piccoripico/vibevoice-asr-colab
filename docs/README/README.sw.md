# VibeVoice-ASR Colab Daftari

Daftari hii inanukuu faili za sauti au video.

## Vipengele

- **Bila malipo kwa matumizi:** Daftari hili linatumia [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), ambayo Microsoft ilifungua chanzo mnamo Januari 21, 2026, pamoja na Google Colab bila malipo GPU wakati wa utekelezaji. Ili kufanya kielelezo kiweze kutumika kwenye GPU zisizolipishwa, daftari hili linatumia kielelezo kilichokadiriwa cha VibeVoice-ASR.
- **Hakuna kikomo kisichobadilika cha ukubwa wa faili ya upande wa daftari:** Faili kubwa za sauti na video zinaweza kuchakatwa kwa kuzigawanya katika sehemu.
- **Uchakataji wa kundi:** Faili nyingi za sauti na video zinaweza kuchakatwa kwa muda mmoja.
- **Google Drive msaada:** Faili zinaweza kupakiwa moja kwa moja, au faili za sauti/video zilizohifadhiwa katika Google Drive zinaweza kuchakatwa.
- **Mkondoni kikamilifu:** Upakuaji na uchakataji unafanywa kwenye Google Colab. Hakuna rasilimali za kompyuta za ndani zinazotumika.
- **Miundo ya towe nyingi:** Daftari hutoa manukuu asili ya JSON na kutolewa TXT, CSV, Markdown, Excel, na SRT faili za manukuu.

## Mpangilio wa Hifadhi

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Daftari ya kiolezo ndio daftari chanzo na ina vishikilia nafasi kama vile
`{{settings.input_mode_help}}`. Faili za eneo hutoa thamani za lugha mahususi
kwa nakala inayoangalia Colab kama vile `# @title`, `# @markdown`, maoni, na kuchaguliwa
ujumbe wa mtumiaji. Kila eneo lina kishikilia nafasi kamili kutoka kwa
Lugha ya asili ya Kiingereza.

Daftari zinazozalishwa zimejitolea ili viungo vya Colab vibaki thabiti.

## Jenga

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Onyesha upya Tafsiri

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` mashine-hutafsiri lugha kamili ya Kiingereza hadi kwenye
faili zingine za eneo wakati wa kuhifadhi ishara za nambari kama vile `INPUT_MODE`,
`Google Drive`, `# @markdown`, na vitambulishi vya miundo. Kagua nakala iliyotafsiriwa
kabla ya kutolewa rasmi.

## Hundi

```bash
python scripts/check_notebooks.py
```

Ukaguzi wa mtiririko wa kazi wa CI kuwa madaftari yaliyotolewa yamesasishwa, hayajahifadhiwa
matokeo ya seli, changanua kama JSON, hayana vishikilia nafasi ambavyo havijatatuliwa, na msimbo huo
seli ni syntax halali ya Python.

## Nini CI Inaweza Kujaribu

Hifadhi hii inaweza kujaribu daftari kama vizalia vya programu vinavyoweza kusambazwa:

- kizazi kutoka kwa kiolezo na faili za eneo
- daftari JSON uhalali
- matokeo tupu ya utekelezaji
- Syntax ya Python katika seli za nambari
- siri za bahati mbaya
- madaftari yaliyotolewa yaliyojitolea yanayolingana na kiolezo

Njia kamili ya unukuzi inahitaji API maalum za Colab, upakuaji wa miundo mikubwa,
GPU upatikanaji, na faili za midia zinazotolewa na mtumiaji. Hiyo inapaswa kutibiwa kama a
jaribio la mwongozo la moshi au mtiririko tofauti wa uthibitishaji wa GPU/Colab.
