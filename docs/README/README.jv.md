# VibeVoice-ASR Colab Notebook

Notebook iki nerjemahake file audio utawa video.

## Fitur

- **Gratis kanggo nggunakake:** Notebook iki nggunakake [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), sing Microsoft open-sourced tanggal 21 Januari 2026, bebarengan karo Google Colab gratis GPU runtime. Kanggo nggawe model bisa digunakake ing GPU gratis, notebook iki nggunakake model VibeVoice-ASR kuantitatif.
- **Ora ana watesan ukuran file sisih notebook sing tetep:** File audio lan video sing gedhe bisa diproses kanthi dibagi dadi bagean.
- **Batch Processing: ** Multiple file audio lan video bisa diproses ing siji roto.
- **Google Drive support:** File bisa langsung diunggah, utawa file audio/video sing disimpen ing Google Drive bisa diproses.
- **Sing online:** Download lan pangolahan ditindakake ing Google Colab. Ora ana sumber daya komputasi lokal sing digunakake.
- **Maneka format output:** Notebook ngasilake transkrip JSON asli lan diturunake TXT, CSV, Markdown, Excel, lan SRT file subtitle.

## Layout Repository

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Notebook template minangka notebook sumber lan ngemot placeholder kayata
`{{settings.input_mode_help}}`. File lokal nyedhiyakake nilai khusus basa
kanggo salinan madhep Colab kayata `# @title`, `# @markdown`, komentar, lan dipilih
pesen pangguna. Saben lokal ngemot set placeholder lengkap saka
lokal sumber Inggris.

Notebook sing digawe digawe supaya tautan Colab tetep stabil.

## Mbangun

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Refresh Terjemahan

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` machine-nerjemahake lokal Inggris lengkap menyang
file lokal liyane nalika njaga token kode kayata `INPUT_MODE`,
`Google Drive`, `# @markdown`, lan pengenal model. Priksa salinan terjemahan
sadurunge release resmi.

## Priksa

```bash
python scripts/check_notebooks.py
```

Alur kerja CI mriksa manawa notebook sing digawe isih anyar, ora disimpen
output sel, parse minangka JSON, ora ngemot placeholder sing ora ditanggulangi, lan kode kasebut
sel sing bener sintaksis Python.

## Apa CI Bisa Test

Repositori iki bisa nguji notebook minangka artefak sing bisa disebarake:

- generasi saka file cithakan lan lokal
- notebook JSON validitas
- output eksekusi kosong
- Sintaks Python ing sel kode
- rahasia sengaja ketok
- notebook kui setya cocog cithakan

Jalur transkripsi lengkap mbutuhake API khusus Colab, download model gedhe,
GPU kasedhiyan, lan file media sing disedhiyakake pangguna. Sing kudu dianggep minangka a
test kumelun manual utawa kapisah GPU/alur kerja validasi Colab.
