# VibeVoice-ASR Colab筆記本

該筆記本轉錄音訊或視訊檔案。

## 特點

- **免費使用：** 此筆記本使用 Microsoft 於 2026 年 1 月 21 日開源的 [VibeVoice-ASR](https://github.com/microsoft/VibeVoice)，以及 Google Colab 的免費 GPU 運行時。為了使模型可在免費 GPU 上使用，此筆記本使用量化VibeVoice-ASR 模型。
- **沒有固定的筆記本端檔案大小限制：** 大的音訊和視訊檔案可以透過將其分割成片段來處理。
- **批次：** 一次可以處理多個音訊和視訊檔案。
- **Google Drive支援：** 可以直接上傳文件，也可以處理Google Drive中儲存的音訊/視訊檔案。
- **完全在線：** 下載和處理在Google Colab 上執行。不使用本地運算資源。
- **多種輸出格式：** 筆記本輸出原始JSON成績單和衍生的TXT、CSV、Markdown、Excel和SRT字幕檔。

## 儲存庫佈局

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

範本筆記本是來源筆記本，包含佔位符，例如
`{{settings.input_mode_help}}`。區域設定檔提供特定於語言的值
對於針對 Colab 的副本，例如 `# @title`、`# @markdown`、評論和選定內容
用戶訊息。每個語言環境都包含完整的佔位符集
英語源語言環境。

生成的筆記本已提交，因此 Colab 連結保持穩定。

## 建造

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## 重新整理翻譯

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` 機器將完整的英語語言環境翻譯成
其他語言環境文件，同時保留代碼標記，例如`INPUT_MODE`，
`Google Drive`、`# @markdown` 和型號識別碼。審閱翻譯副本
在正式發布之前。

## 支票

```bash
python scripts/check_notebooks.py
```

CI 工作流程檢查產生的筆記本是否是最新的，沒有儲存
單元格輸出，解析為JSON，不包含未解析的佔位符，且該程式碼
cells 是有效的 Python 語法。

## CI 可以測試什麼

此儲存庫可以將筆記本作為可分發工件進行測試：

- 從模板和語言環境文件生成
- 筆記本JSON有效期限
- 空執行輸出
- 程式碼單元中的 Python 語法
- 明顯的意外秘密
- 提交的生成的與模板匹配的筆記本

完整的轉錄路徑需要 Colab 特定的 API、大型模型下載、
GPU 可用性以及用戶提供的媒體檔案。這應該被視為
手動冒煙測試或單獨的 GPU/Colab 驗證工作流程。
