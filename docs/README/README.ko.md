# VibeVoice-ASR Colab 노트북

이 노트북은 오디오 또는 비디오 파일을 기록합니다.

## 기능

- **무료 사용:** 이 노트북은 Google Colab의 무료 GPU 런타임과 함께 Microsoft가 2026년 1월 21일에 오픈 소스로 제공한 [VibeVoice-ASR](https://github.com/microsoft/VibeVoice)을 사용합니다. 모델을 무료 GPU에서 사용할 수 있도록 하기 위해 이 노트북은 양자화된 VibeVoice-ASR 모델을 사용합니다.
- **고정된 노트북 측 파일 크기 제한 없음:** 대용량 오디오 및 비디오 파일을 세그먼트로 분할하여 처리할 수 있습니다.
- **일괄 처리:** 여러 오디오 및 비디오 파일을 한 번에 처리할 수 있습니다.
- **Google Drive 지원:** 파일을 직접 업로드하거나 Google Drive에 저장된 오디오/비디오 파일을 처리할 수 있습니다.
- **완전한 온라인:** 다운로드 및 처리는 Google Colab에서 수행됩니다. 로컬 컴퓨팅 리소스는 사용되지 않습니다.
- **다양한 출력 형식:** 노트북은 원본 JSON 대본과 파생된 TXT, CSV, Markdown, Excel 및 SRT 자막 파일을 출력합니다.

## 저장소 레이아웃

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

템플릿 노트북은 소스 노트북이며 다음과 같은 자리 표시자를 포함합니다.
`{{settings.input_mode_help}}`. 로케일 파일은 언어별 값을 제공합니다.
`# @title`, `# @markdown`, 댓글 및 선택된 Colab용 사본의 경우
사용자 메시지. 모든 로케일에는
영어 소스 로케일.

생성된 노트북은 커밋되므로 Colab 링크는 안정적으로 유지됩니다.

## 짓다

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## 번역 새로 고침

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` 전체 영어 로케일을 기계 번역합니다.
`INPUT_MODE`과 같은 코드 토큰을 유지하면서 다른 로케일 파일
`Google Drive`, `# @markdown` 및 모델 식별자입니다. 번역본 검토
정식 출시 전.

## 체크 무늬

```bash
python scripts/check_notebooks.py
```

CI 워크플로는 생성된 노트북이 최신 상태인지, 저장된 노트북이 없는지 확인합니다.
셀 출력, JSON으로 구문 분석, 해결되지 않은 자리 표시자를 포함하지 않음, 해당 코드
셀은 유효한 Python 구문입니다.

## CI가 테스트할 수 있는 것

이 저장소는 노트북을 배포 가능한 아티팩트로 테스트할 수 있습니다.

- 템플릿 및 로케일 파일에서 생성
- 노트북 JSON 유효성
- 빈 실행 출력
- 코드 셀의 Python 구문
- 명백한 우발적 비밀
- 템플릿과 일치하는 커밋된 생성된 노트북

전체 전사 경로에는 Colab 관련 API, 대규모 모델 다운로드,
GPU 가용성 및 사용자 제공 미디어 파일. 그것은 다음과 같이 취급되어야 한다.
수동 연기 테스트 또는 별도의 GPU/Colab 검증 워크플로.
