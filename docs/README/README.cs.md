# VibeVoice-ASR Zápisník Colab

Tento notebook přepisuje audio nebo video soubory.

## Funkce

- **Zdarma k použití:** Tento notebook používá [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), kterou společnost Microsoft získala jako open source 21. ledna 2026, spolu s bezplatným runtime GPU od Google Colab. Aby byl model použitelný na bezplatných GPU, tento notebook používá kvantovaný model VibeVoice-ASR.
- **Žádný pevný limit velikosti souborů na straně notebooku:** Velké audio a video soubory lze zpracovat jejich rozdělením na segmenty.
- **Dávkové zpracování:** V jednom běhu lze zpracovat více audio a video souborů.
- Podpora **Google Drive:** Soubory lze nahrát přímo nebo lze zpracovat audio/video soubory uložené v Google Drive.
- **Plně online:** Stahování a zpracování se provádí na Google Colab. Nejsou používány žádné místní výpočetní zdroje.
- **Více výstupních formátů:** Výstupem notebooku je původní přepis JSON a odvozené soubory s titulky TXT, CSV, Markdown, Excel a SRT.

## Rozložení úložiště

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Šablona poznámkového bloku je zdrojovým poznámkovým blokem a obsahuje zástupné symboly jako např
`{{settings.input_mode_help}}`. Soubory národního prostředí poskytují hodnoty specifické pro jazyk
pro kopii na straně Colab, například `# @title`, `# @markdown`, komentáře a vybrané
uživatelské zprávy. Každé národní prostředí obsahuje úplnou sadu zástupných symbolů z
Anglické zdrojové národní prostředí.

Vygenerované poznámkové bloky se zavazují, takže odkazy Colab zůstávají stabilní.

## Vytvořit

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Obnovit překlady

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` stroj-přeloží plné anglické národní prostředí do
další soubory národního prostředí při zachování tokenů kódu, jako je `INPUT_MODE`,
`Google Drive`, `# @markdown` a identifikátory modelu. Zkontrolovat přeloženou kopii
před oficiálním propuštěním.

## Kontroly

```bash
python scripts/check_notebooks.py
```

Pracovní postup CI kontroluje, zda jsou vygenerované poznámkové bloky aktuální, nejsou uloženy
výstupy buněk, analyzujte jako JSON, neobsahují žádné nevyřešené zástupné symboly a tento kód
buňky jsou platnou syntaxí Pythonu.

## Co může CI testovat

Toto úložiště může otestovat notebook jako distribuovatelný artefakt:

- generování ze souborů šablony a národního prostředí
- notebook JSON platnost
- prázdné výstupy provádění
- Syntaxe Pythonu v buňkách kódu
- zjevná náhodná tajemství
- potvrzené vygenerované poznámkové bloky odpovídající šabloně

Úplná cesta přepisu vyžaduje rozhraní API specifická pro Colab, stahování velkých modelů,
Dostupnost GPU a multimediální soubory poskytované uživatelem. To by mělo být považováno za a
manuální kouřový test nebo samostatný pracovní postup ověření GPU/Colab.
