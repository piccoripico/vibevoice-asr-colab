# VibeVoice-ASR Buku Nota Colab

Buku nota ini menyalin fail audio atau video.

## Ciri-ciri

- **Percuma untuk digunakan:** Buku nota ini menggunakan [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), yang Microsoft sumber terbuka pada 21 Januari 2026, bersama-sama dengan Google Colab masa jalan GPU percuma. Untuk menjadikan model itu boleh digunakan pada GPU percuma, buku nota ini menggunakan model VibeVoice-ASR terkuantisasi.
- **Tiada had saiz fail sisi buku nota tetap:** Fail audio dan video yang besar boleh diproses dengan membahagikannya kepada segmen.
- **Pemprosesan kelompok:** Berbilang fail audio dan video boleh diproses dalam satu larian.
- **Google Drive sokongan:** Fail boleh dimuat naik terus, atau fail audio/video yang disimpan dalam Google Drive boleh diproses.
- **Dalam talian sepenuhnya:** Muat turun dan pemprosesan dilakukan pada Google Colab. Tiada sumber pengkomputeran tempatan digunakan.
- **Berbilang format output:** Buku nota mengeluarkan transkrip JSON asal dan terbitan TXT, CSV, Markdown, Excel dan SRT fail sari kata.

## Susun atur Repositori

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Buku nota templat ialah buku nota sumber dan mengandungi ruang letak seperti
`{{settings.input_mode_help}}`. Fail setempat menyediakan nilai khusus bahasa
untuk salinan menghadap Colab seperti `# @title`, `# @markdown`, ulasan dan dipilih
mesej pengguna. Setiap tempat mengandungi set pemegang tempat lengkap daripada
Tempatan sumber Inggeris.

Buku nota yang dijana adalah komited supaya pautan Colab kekal stabil.

## bina

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Muat semula Terjemahan

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` mesin-menerjemahkan tempat bahasa Inggeris penuh ke dalam
fail setempat lain sambil mengekalkan token kod seperti `INPUT_MODE`,
`Google Drive`, `# @markdown` dan pengecam model. Semak salinan terjemahan
sebelum keluaran rasmi.

## Cek

```bash
python scripts/check_notebooks.py
```

Semakan aliran kerja CI bahawa buku nota yang dijana adalah terkini, tidak disimpan
output sel, menghuraikan sebagai JSON, tidak mengandungi ruang letak yang tidak diselesaikan dan kod itu
sel adalah sintaks Python yang sah.

## Apa yang CI Boleh Uji

Repositori ini boleh menguji buku nota sebagai artifak boleh diagihkan:

- penjanaan daripada templat dan fail setempat
- buku nota JSON kesahan
- output pelaksanaan kosong
- Sintaks Python dalam sel kod
- rahsia tidak sengaja yang jelas
- buku nota terjana komited yang sepadan dengan templat

Laluan transkripsi penuh memerlukan API khusus Colab, muat turun model besar,
GPU ketersediaan, dan fail media yang disediakan pengguna. Itu harus dianggap sebagai a
ujian asap manual atau aliran kerja pengesahan GPU/Colab yang berasingan.
