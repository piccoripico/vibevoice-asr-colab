# VibeVoice-ASR Блокнот Colab

Этот блокнот позволяет расшифровывать аудио- и видеофайлы.

## Особенности

- **Бесплатно для использования.** В этом ноутбуке используется [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), исходный код которого Microsoft открыла 21 января 2026 г., а также бесплатная среда выполнения GPU для Google Colab. Чтобы модель можно было использовать на бесплатных графических процессорах, в этом ноутбуке используется квантованная модель VibeVoice-ASR.
- **Нет фиксированного ограничения на размер файла на стороне ноутбука.** Большие аудио- и видеофайлы можно обрабатывать, разделяя их на сегменты.
- **Пакетная обработка:** за один проход можно обработать несколько аудио- и видеофайлов.
- Поддержка **Google Drive:** файлы можно загружать напрямую или обрабатывать аудио/видео файлы, хранящиеся в Google Drive.
- **Полностью онлайн:** Загрузка и обработка выполняются на Google Colab. Никакие локальные вычислительные ресурсы не используются.
- **Несколько форматов вывода:** Ноутбук выводит исходную расшифровку JSON и производные файлы субтитров TXT, CSV, Markdown, Excel и SRT.

## Макет репозитория

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Блокнот-шаблон является исходным блокнотом и содержит заполнители, такие как
`{{settings.input_mode_help}}`. Файлы локали предоставляют значения, специфичные для языка.
для копий с интерфейсом Colab, таких как `# @title`, `# @markdown`, комментарии и выбранные
сообщения пользователей. Каждая локаль содержит полный набор заполнителей из
Английская локаль источника.

Созданные блокноты фиксируются, поэтому ссылки Colab остаются стабильными.

## Строить

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Обновить переводы

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` автоматически переводит полную английскую локаль в
другие файлы локали, сохраняя при этом токены кода, такие как `INPUT_MODE`,
`Google Drive`, `# @markdown` и идентификаторы моделей. Проверить переведенную копию
до официального релиза.

## Чеки

```bash
python scripts/check_notebooks.py
```

Рабочий процесс CI проверяет актуальность созданных записных книжек и не сохраняет их.
выходные данные ячеек, анализируемые как JSON, не содержат неразрешенных заполнителей, и этот код
ячейки являются допустимым синтаксисом Python.

## Что может тестировать CI

Этот репозиторий может протестировать записную книжку как распространяемый артефакт:

- генерация из файлов шаблона и локали
- срок действия блокнота JSON
- пустые выходные данные выполнения
- Синтаксис Python в ячейках кода
- очевидные случайные секреты
- зафиксированные созданные блокноты, соответствующие шаблону

Полный путь транскрипции требует API-интерфейсов Colab, больших загрузок моделей,
Доступность GPU и медиафайлы, предоставленные пользователем. К этому следует относиться как к
ручное дымовое тестирование или отдельный рабочий процесс проверки GPU/Colab.
