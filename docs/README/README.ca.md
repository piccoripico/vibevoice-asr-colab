# VibeVoice-ASR Llibreta Colab

Aquest quadern transcriu fitxers d'àudio o vídeo.

## Característiques

- **Gratis d'utilitzar:** aquest quadern utilitza [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), que Microsoft va crear de codi obert el 21 de gener de 2026, juntament amb el temps d'execució gratuït GPU de Google Colab. Per fer que el model es pugui utilitzar a les GPU gratuïtes, aquest quadern utilitza un model VibeVoice-ASR quantificat.
- **No hi ha límit fix de mida de fitxer del costat del portàtil:** Els fitxers d'àudio i vídeo grans es poden processar dividint-los en segments.
- **Processament per lots:** Es poden processar diversos fitxers d'àudio i vídeo en una execució.
- Suport **Google Drive:** Els fitxers es poden carregar directament o els fitxers d'àudio/vídeo emmagatzemats a Google Drive es poden processar.
- **Tot en línia:** Les descàrregues i el processament es realitzen a Google Colab. No s'utilitzen recursos informàtics locals.
- **Múltiples formats de sortida:** el quadern genera la transcripció original JSON i els fitxers de subtítols TXT, CSV, Markdown, Excel i SRT derivats.

## Disseny del dipòsit

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

El quadern de plantilla és el quadern d'origen i conté marcadors de posició com ara
`{{settings.input_mode_help}}`. Els fitxers de configuració regional proporcionen valors específics de l'idioma
per a còpies orientades a Colab, com ara `# @title`, `# @markdown`, comentaris i seleccionats
missatges dels usuaris. Cada configuració regional conté el conjunt complet de marcadors de posició de l'
Localització d'origen anglès.

Els quaderns generats estan compromesos perquè els enllaços de Colab es mantinguin estables.

## Construir

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Actualitzar les traduccions

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` tradueix automàticament la configuració regional completa de l'anglès al
altres fitxers de configuració regional conservant els testimonis de codi com ara `INPUT_MODE`,
`Google Drive`, `# @markdown` i identificadors del model. Revisa la còpia traduïda
abans d'un alliberament formal.

## xecs

```bash
python scripts/check_notebooks.py
```

Les comprovacions del flux de treball de CI que els quaderns generats estan actualitzades, no s'han desat
Les sortides de cel·les, analitzades com a JSON, no contenen marcadors de posició no resolts i aquest codi
les cel·les són una sintaxi de Python vàlida.

## Què pot provar CI

Aquest dipòsit pot provar el quadern com a artefacte distribuïble:

- generació a partir de la plantilla i els fitxers de configuració regional
- quadern JSON validesa
- sortides d'execució buides
- Sintaxi de Python a les cel·les de codi
- secrets accidentals evidents
- quaderns generats compromesos que coincideixen amb la plantilla

La ruta de transcripció completa requereix API específiques de Colab, descàrregues de models grans,
Disponibilitat de GPU i fitxers multimèdia proporcionats per l'usuari. Això s'ha de tractar com a
prova de fum manual o un flux de treball de validació GPU/Colab independent.
