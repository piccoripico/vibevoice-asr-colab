# VibeVoice-ASR Блокнот Colab

Цей блокнот транскрибує аудіо- та відеофайли.

## Особливості

- **Безкоштовне використання:** Цей ноутбук використовує [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), який Microsoft відкрила 21 січня 2026 року, разом із Google Colab безкоштовним середовищем виконання GPU. Щоб цю модель можна було використовувати на безкоштовних графічних процесорах, цей ноутбук використовує квантовану модель VibeVoice-ASR.
- **Немає фіксованого обмеження розміру файлу на стороні блокнота:** Великі аудіо- та відеофайли можна обробляти, розділяючи їх на сегменти.
- **Пакетна обробка:** кілька аудіо- та відеофайлів можна обробити за один запуск.
- **Google Drive підтримка: ** Файли можна завантажувати безпосередньо або аудіо/відеофайли, що зберігаються в Google Drive, можна обробляти.
- **Повністю онлайн:** Завантаження та обробка виконуються на Google Colab. Локальні обчислювальні ресурси не використовуються.
- **Кілька вихідних форматів:** Ноутбук виводить оригінальну транскрипцію JSON і похідні файли субтитрів TXT, CSV, Markdown, Excel і SRT.

## Макет сховища

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Шаблон блокнота є вихідним блокнотом і містить заповнювачі, як-от
`{{settings.input_mode_help}}`. Локальні файли надають значення для певної мови
для копії, що відкривається для Colab, як-от `# @title`, `# @markdown`, коментарі та вибрані
повідомлення користувачів. Кожна локаль містить повний набір заповнювачів із
Англійська вихідна мова.

Згенеровані блокноти зафіксовано, тому посилання Colab залишаються стабільними.

## Будувати

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Оновити переклади

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` машинно перекладає повну англійську мову на
інші файли локалізації, зберігаючи маркери коду, такі як `INPUT_MODE`,
`Google Drive`, `# @markdown` та ідентифікатори моделі. Перегляньте перекладену копію
до офіційного звільнення.

## Чеки

```bash
python scripts/check_notebooks.py
```

Робочий процес CI перевіряє, чи згенеровані блокноти актуальні та не збережені
вихідні дані комірки, аналізовані як JSON, не містять невирішених заповнювачів, і цей код
клітинки мають дійсний синтаксис Python.

## Що CI може перевірити

Це сховище може тестувати блокнот як розповсюджуваний артефакт:

- створення з файлів шаблону та локалі
- термін дії зошита JSON
- порожні виходи виконання
- Синтаксис Python у клітинках коду
- явні випадкові секрети
- committed створені блокноти, що відповідають шаблону

Повний шлях транскрипції вимагає API-інтерфейсів Colab, великих завантажень моделей,
GPU доступність і медіафайли, надані користувачем. Це слід розглядати як a
ручний димовий тест або окремий робочий процес перевірки GPU/Colab.
