# VibeVoice-ASR Colab Bilježnica

Ova bilježnica transkribira audio ili video datoteke.

## Značajke

- **Besplatno korištenje:** Ovo prijenosno računalo koristi [VibeVoice-ASR](https://github.com/microsoft/VibeVoice), koji je Microsoft otvorio 21. siječnja 2026., zajedno s Google Colab besplatnim GPU runtimeom. Kako bi model bio upotrebljiv na besplatnim GPU-ovima, ovo prijenosno računalo koristi kvantizirani VibeVoice-ASR model.
- **Nema fiksnog ograničenja veličine datoteke na strani prijenosnog računala:** Velike audio i video datoteke mogu se obraditi njihovim dijeljenjem u segmente.
- **Skupna obrada:** Više audio i video datoteka može se obraditi u jednom izvođenju.
- **Google Drive podrška:** Datoteke se mogu izravno učitati ili se audio/video datoteke pohranjene u Google Drive mogu obraditi.
- **Potpuno online:** Preuzimanja i obrada izvode se na Google Colab. Ne koriste se lokalni računalni resursi.
- **Višestruki izlazni formati:** Prijenosno računalo ispisuje izvorni JSON transkript i izvedene TXT, CSV, Markdown, Excel i SRT datoteke titlova.

## Izgled repozitorija

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Bilježnica predloška je izvorna bilježnica i sadrži rezervirana mjesta kao što su
`{{settings.input_mode_help}}`. Lokalne datoteke daju vrijednosti specifične za jezik
za kopiju okrenutu prema Colabu kao što su `# @title`, `# @markdown`, komentari i odabrani
korisničke poruke. Svaka lokalizacija sadrži kompletan skup rezerviranih mjesta iz
Engleski izvorni jezik.

Generirane bilježnice predane su tako da Colab veze ostaju stabilne.

## Izgraditi

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Osvježi prijevode

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` strojno prevodi puni engleski jezik u
druge lokalne datoteke uz očuvanje tokena koda kao što je `INPUT_MODE`,
`Google Drive`, `# @markdown` i identifikatori modela. Pregledajte prevedenu kopiju
prije službenog puštanja na slobodu.

## Čekovi

```bash
python scripts/check_notebooks.py
```

CI tijek rada provjerava jesu li generirane bilježnice ažurne, nema spremljenih
izlazi ćelija, raščlaniti kao JSON, ne sadrže nerazriješena rezervirana mjesta i taj kod
ćelije su važeća Python sintaksa.

## Što CI može testirati

Ovo spremište može testirati bilježnicu kao artefakt koji se može distribuirati:

- generiranje iz datoteka predloška i lokalizacije
- bilježnica JSON valjanost
- prazni izvodi izvršenja
- Python sintaksa u ćelijama koda
- očite slučajne tajne
- committed generirane bilježnice koje odgovaraju predlošku

Put pune transkripcije zahtijeva API-je specifične za Colab, velika preuzimanja modela,
GPU dostupnost i medijske datoteke koje dostavlja korisnik. To treba tretirati kao
ručni test dima ili zasebni GPU/Colab tijek rada provjere valjanosti.
