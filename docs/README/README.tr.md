# VibeVoice-ASR Colab Defter

Bu not defteri ses veya video dosyalarının yazıya geçirilmesini sağlar.

## Özellikler

- **Kullanımı ücretsiz:** Bu dizüstü bilgisayar, Microsoft'un 21 Ocak 2026'da açık kaynak sağladığı [VibeVoice-ASR](https://github.com/microsoft/VibeVoice) ile birlikte Google Colab'nin ücretsiz GPU çalışma zamanını kullanır. Modeli ücretsiz GPU'larda kullanılabilir hale getirmek için bu dizüstü bilgisayar, nicelenmiş bir VibeVoice-ASR modelini kullanır.
- **Dizüstü bilgisayar tarafında sabit dosya boyutu sınırı yoktur:** Büyük ses ve video dosyaları, bölümlere bölünerek işlenebilir.
- **Toplu işleme:** Tek seferde birden fazla ses ve video dosyası işlenebilir.
- **Google Drive desteği:** Dosyalar doğrudan yüklenebilir veya Google Drive içinde saklanan ses/video dosyaları işlenebilir.
- **Tamamen çevrimiçi:** İndirmeler ve işlemler Google Colab üzerinden gerçekleştirilir. Hiçbir yerel bilgi işlem kaynağı kullanılmaz.
- **Birden fazla çıktı formatı:** Dizüstü bilgisayar, orijinal JSON transkripsiyonunu ve türetilmiş TXT, CSV, Markdown, Excel ve SRT altyazı dosyalarının çıktısını verir.

## Depo Düzeni

```text
notebook_template/
  VibeVoice_ASR_Colab.template.ipynb
locales/
  *.json
notebooks/
  VibeVoice_ASR_Colab_<language-code>.ipynb
docs/
  README/
    README.<language-code>.md
scripts/
  build_readmes.py
  build_notebooks.py
  check_notebooks.py
  translate_full_locales.py
```

Şablon not defteri kaynak not defteridir ve aşağıdaki gibi yer tutucuları içerir:
`{{settings.input_mode_help}}`. Yerel ayar dosyaları dile özgü değerler sağlar
`# @title`, `# @markdown` gibi Colab'a yönelik kopyalar, yorumlar ve seçilenler için
kullanıcı mesajları. Her yerel ayar, yer tutucu setinin tamamını içerir.
İngilizce kaynak yerel ayarı.

Colab bağlantılarının sabit kalması için oluşturulan not defterleri kaydedilir.

## İnşa etmek

```bash
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Çevirileri Yenile

```bash
python scripts/translate_full_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
python scripts/check_notebooks.py
```

`translate_full_locales.py` tam İngilizce yerel ayarını makineyle çevirir
`INPUT_MODE` gibi kod belirteçlerini korurken diğer yerel ayar dosyalarını,
`Google Drive`, `# @markdown` ve model tanımlayıcıları. Çevrilen kopyayı inceleyin
resmi bir açıklama öncesinde.

## Çekler

```bash
python scripts/check_notebooks.py
```

Oluşturulan not defterlerinin güncel olup olmadığını ve kayıtlı not defterlerinin bulunmadığını CI iş akışı kontrolleri
JSON olarak ayrıştırılan hücre çıktıları, çözümlenmemiş yer tutucular içermez ve bu kod
hücreler geçerli Python sözdizimidir.

## CI Neleri Test Edebilir?

Bu depo, not defterini dağıtılabilir bir yapı olarak test edebilir:

- şablon ve yerel ayar dosyalarından oluşturma
- defter JSON geçerliliği
- boş yürütme çıktıları
- Kod hücrelerinde Python sözdizimi
- bariz tesadüfi sırlar
- şablonla eşleşen taahhüt edilmiş oluşturulmuş not defterleri

Tam transkripsiyon yolu için Colab'a özel API'ler, büyük model indirmeleri,
GPU kullanılabilirliği ve kullanıcı tarafından sağlanan medya dosyaları. Bu bir şey olarak ele alınmalı
manuel duman testi veya ayrı bir GPU/Colab doğrulama iş akışı.
